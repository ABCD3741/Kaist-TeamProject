## 액션 아이템 1
- 사용자 액션: 건의사항 게시글을 작성한다
- 서버가 해야 할 일: 새 건의사항 게시글을 저장한다
- 저장할 데이터: 제목, 내용, 작성자명
- 후보 테이블: suggestions

## 액션 아이템 2
- 사용자 액션: 건의사항 게시글 목록을 조회한다
- 서버가 해야 할 일: 건의사항 게시글 목록을 좋아요 순으로 가져온다
- 저장할 데이터: 저장된 건의사항 게시글
- 후보 테이블: suggestions

## 액션 아이템 3
- 사용자 액션: 건의사항 게시글을 삭제한다
- 서버가 해야 할 일: 특정 건의사항 게시글을 삭제한다
- 저장할 데이터: 게시글 id
- 후보 테이블: suggestions

## 액션 아이템 4
- 사용자 액션: 건의사항 게시글에 좋아요를 누른다
- 서버가 해야 할 일: 특정 건의사항 게시글의 좋아요를 저장한다
- 저장할 데이터: 건의사항 게시글 id, 사용자명
- 후보 테이블: likes

## 액션 아이템 5
- 사용자 액션: 건의사항 게시글의 좋아요를 취소한다
- 서버가 해야 할 일: 특정 건의사항 게시글의 좋아요를 삭제한다
- 저장할 데이터: 좋아요 id
- 후보 테이블: likes
